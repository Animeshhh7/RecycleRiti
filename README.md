# ♻️ RecycleRiti

A waste management app to schedule pickups, track agents in real-time, engage in eco-friendly community activities, and earn rewards.

---

## 🛠️ Tech Stack
- Flutter  
- Node.js (Express.js)  
- PostgreSQL  

---

## 🚀 Features
- Schedule recycling pickups  
- Live agent tracking  
- Notifications & updates  
- Pickup confirmation  
- Join nearby events  
- User & agent dashboards

---

## ⚙️ How It Works
1. Sign up  
2. Schedule pickup (date & time)  
3. Agent accepts request  
4. Track pickup live  
5. Pickup completed & confirmed
